# Api Golang

## Clone

```
git clone git@github.com:Pietrucci-Blacher/apigolang
```

## Init the database

run this command
```bash
docker compose up -d
```
and import this file `go_api.sql` in [adminer](http://localhost:8080)

## Link

[Api](http://localhost:8081/)
[Swagger](http://localhost:8081/api/swagger/index.html)
